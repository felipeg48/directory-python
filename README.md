# Directory Exercise

Start by clone this repository:
```shell script
git clone https://github.com/felipeg48/directory-python
```

## Server

This is the backend.

```shell script
cd server
python3.8 -m venv venv
source venv/bin/activate
pip install requirements.txt
FLASK_APP=app.py flask run
```
